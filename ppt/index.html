技术核心 · TOOL CALLING

用 LLM 自带的工具调用，把成本压到 1/10。

工具调用 = 结构化输出
让 LLM 通过 tool_calls 返回 JSON，而非自由文本。省去解析、重试、纠错，token 利用率最高。
STRUCTURED · NOT FREE TEXT
一次调用，全套产出
同一次 LLM 调用里：分句 + 翻译 + 拆词 + 词性 + 音标 + 例句 + 语法。不要重复合令。
ONE CALL · ALL OUTPUTS
程序只做编排
LLM 吐 JSON，程序存储 + 跳转 + 渲染。两者职责清晰，架构极简。
